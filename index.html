Linux Terminal Simulator

Experience a fully featured Linux-like terminal right in your browser

About

This project simulates a Linux terminal with multiple tabs, a persistent in-browser file system, command autocompletion, command history, a built-in basic file editor, syntax highlighting, and much more. Ideal for learning, experimentation, and fun!

Features

  • Multiple independent terminal tabs with separate sessions
  • Persistent simulated Linux file system with file and folder management
  • Command autocompletion using the Tab key
  • Basic `nano` text editor within the terminal
  • Keyboard command history navigation
  • Syntax highlighted terminal output and commands
  • Mouse support for tab switching and text selection
  • Customizable command aliases
  • Realistic Linux command experience, all in the browser
